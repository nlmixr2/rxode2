# Return if the object can be stacked

Return if the object can be stacked

## Usage

``` r
is.rxStackData(object)
```

## Arguments

- object:

  object to test if it can be stacked

## Value

boolean to tell if an object can be stacked using rxode2

## Author

Matthew L. Fidler

## Examples

``` r
is.rxStackData(NULL)
#> [1] FALSE
```
