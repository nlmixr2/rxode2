# Assert a variable would be new to the model

Assert a variable would be new to the model

## Usage

``` r
assertVariableNew(ui, x)
```

## Arguments

- ui:

  rxode2 ui model

- x:

  would the variable `x` variable be new in the model

## Value

nothing, but will error if `x` would not be new
