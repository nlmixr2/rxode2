# Check to see if this is an rxSolve object.

Check to see if this is an rxSolve object.

## Usage

``` r
is.rxSolve(x)
```

## Arguments

- x:

  object to check to see if it is rxSolve

  If this is an rxSolve object that has expired strip all rxSolve
  information.

## Value

boolean indicating if this is a `rxSolve` object
