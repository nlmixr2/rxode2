# Use the udf model variable information to get the environment where the functions exists

Use the udf model variable information to get the environment where the
functions exists

## Usage

``` r
.udfEnvSetUdf(udf)
```

## Arguments

- udf:

  modelVars\$udf, integer vector with NA_integer\_ for the address of
  the environment where the functions exist

## Value

nothing called for side effects
