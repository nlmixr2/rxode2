# Lock/Unlock environment for getting R user functions

Lock/Unlock environment for getting R user functions

## Usage

``` r
.udfEnvReset(lock = TRUE)
```

## Arguments

- lock:

  logical to see if environment to look for user defined functions is
  locked. If it is locked then environments are not assigned. When NULL
  returns lock status

## Value

lock status
