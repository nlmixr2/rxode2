# Jacobian and parameter derivatives

Return Jacobain and parameter derivatives

## Usage

``` r
rxDfdy(obj)
```

## Arguments

- obj:

  rxode2 family of objects

## Value

A list of the jacobian parameters defined in this rxode2 object.
