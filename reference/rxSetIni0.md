# Set Initial conditions to time zero instead of the first observed/dosed time

Set Initial conditions to time zero instead of the first observed/dosed
time

## Usage

``` r
rxSetIni0(ini0 = TRUE)
```

## Arguments

- ini0:

  When `TRUE` (default), set initial conditions to time zero. Otherwise
  the initial conditions are the first observed time.

## Value

the boolean ini0, though this is called for its side effects
