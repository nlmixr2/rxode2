# rxUiDevelop - Enable/Disable rxUi development. Here all \$ completions are given

rxUiDevelop - Enable/Disable rxUi development. Here all \$ completions
are given

## Usage

``` r
rxUiDevelop(enable = TRUE)
```

## Arguments

- enable:

  boolean to enable/disable rxUi development mode.

## Value

nothing, called for side effects

## Author

Matthew L. Fidler

## Examples

``` r
rxUiDevelop(TRUE)
rxUiDevelop(FALSE)
```
