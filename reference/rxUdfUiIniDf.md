# Get the rxode2 iniDf of the current UI being processed (or return NULL)

Get the rxode2 iniDf of the current UI being processed (or return NULL)

## Usage

``` r
rxUdfUiIniDf()
```

## Value

Initial `data.frame` being processed or `NULL` for nothing.
