# Assign the ini block in the rxode2 related object

Assign the ini block in the rxode2 related object

## Usage

``` r
ini(x, envir = environment(x)) <- value
```

## Arguments

- x:

  rxode2 related object

- envir:

  Environment where assignment occurs

- value:

  Value of the object

## Value

rxode2 related object
