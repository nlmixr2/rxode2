# Convert a Raw Vector or R object to C Code

This function converts a raw vector or R object to C code that is used
in the rxode2 model

## Usage

``` r
rxRawToC(raw, type = c("xz", "qs2", "qdata", "base", "bzip2"))
```

## Arguments

- raw:

  raw vector or R object to convert

- type:

  serialization type; one of "qs2", "qdata", "base", "xz" or "bzip2".

## Value

character string of C code

Matthew L. Fidler
