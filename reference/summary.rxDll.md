# Summary of rxDll object

This gives expanded information about the rxDll object

## Usage

``` r
# S3 method for class 'rxDll'
summary(object, ...)
```

## Arguments

- object:

  RxDll object

- ...:

  Other arguments. Includes `noprint`, which is a logical telling if the
  object should print the rxDll object first. By default this is FALSE

## Value

object is returned
