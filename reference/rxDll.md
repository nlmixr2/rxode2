# Return the DLL associated with the rxode2 object

This will return the dynamic load library or shared object used to run
the C code for rxode2.

## Usage

``` r
rxDll(obj)
```

## Arguments

- obj:

  A rxode2 family of objects or a character string of the model
  specification or location of a file with a model specification.

## Value

a path of the library
