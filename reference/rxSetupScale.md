# Setup the initial conditions.

Setup the initial conditions.

## Usage

``` r
rxSetupScale(obj, scale = NULL, extraArgs = NULL)
```

## Arguments

- obj:

  rxode2 object

- scale:

  A numeric vector scales

- extraArgs:

  A list of extra args to parse for initial conditions.

## Value

setup scale for changing compartment values
