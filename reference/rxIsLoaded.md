# Determine if the DLL associated with the rxode2 object is loaded

Determine if the DLL associated with the rxode2 object is loaded

## Usage

``` r
rxIsLoaded(obj)

rxDllLoaded(obj)
```

## Arguments

- obj:

  A rxode2 family of objects

## Value

Boolean returning if the rxode2 library is loaded.
