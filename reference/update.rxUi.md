# Update for rxUi

Update for rxUi

## Usage

``` r
# S3 method for class 'rxUi'
update(object, ..., envir = parent.frame())
```

## Arguments

- object:

  rxode2 UI object

- ...:

  Lines to update

- envir:

  Environment for evaluating ini() style calls

## Value

a new rxode2 updated UI object
