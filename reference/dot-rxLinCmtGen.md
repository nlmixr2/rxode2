# Internal function to generate the model variables for a linCmt() model

Internal function to generate the model variables for a linCmt() model

## Usage

``` r
.rxLinCmtGen(lenState, vars)
```

## Arguments

- lenState:

  Length of the state

- vars:

  Variables in the model

## Value

Model variables of expanded linCmt model
