# Internal Pruning function

Internal Pruning function

## Usage

``` r
.rxPrune(x, envir = parent.frame(), strAssign = list())
```

## Arguments

- x:

  List of quoted lines

- envir:

  Environment where information is stored

- strAssign:

  string assignment list from model variables

## Value

Pruned model code
