# Clear/Set pipeline

Clear/Set pipeline

## Usage

``` r
.clearPipe(
  rx = NULL,
  inits = NULL,
  events = NULL,
  params = NULL,
  iCov = NULL,
  keep = NULL,
  thetaMat = NULL,
  omega = NULL,
  sigma = NULL,
  dfObs = NULL,
  dfSub = NULL,
  nSub = NULL,
  nStud = NULL
)
```

## Arguments

- rx:

  rxode2 object

## Value

None, clears rxode2 pipeline
