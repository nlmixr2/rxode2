# Return the current estimation method for the UI processing

Return the current estimation method for the UI processing

## Usage

``` r
rxUdfUiEst(value)
```

## Arguments

- value:

  when specified, this assigns the character value of the estimation
  method or NULL if there is nothing being estimated

## Value

value of the estimation method being processed or NULL
