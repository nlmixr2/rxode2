## Usage

``` r
rxEtDispatchSolve(x, ...)

# Default S3 method
rxEtDispatchSolve(x, ...)
```

## Arguments

- x:

  rxode2 solve dispatch object

- ...:

  other arguments

## Value

if 'rxode2' is loaded, a solved object, otherwise an error
