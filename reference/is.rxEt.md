# Check to see if this is an rxEt object.

Check to see if this is an rxEt object.

## Usage

``` r
is.rxEt(x)
```

## Arguments

- x:

  object to check to see if it is rxEt

  If this is an rxEt object that has expired strip all rxEt information.

## Value

Boolean indicating if this is a rxode2 event table
