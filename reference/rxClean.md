# Cleanup anonymous DLLs by unloading them

This cleans up any rxode2 loaded DLLs

## Usage

``` r
rxClean(wd)
```

## Arguments

- wd:

  What directory should be cleaned; (DEPRECIATED), this no longer does
  anything.

  This unloads all rxode2 anonymous dlls.

## Value

TRUE if successful
