# Serialize an R Object to a Raw Vector

Serialize an R Object to a Raw Vector

## Usage

``` r
rxSerialize(x, type = c("qs2", "qdata", "base"))
```

## Arguments

- x:

  object to serialize

- type:

  serialization type; one of "qs2", "qdata", or "base".

## Value

raw vector

Matthew L. Fidler
