# Serialize an R Object to a Raw Vector

Serialize an R Object to a Raw Vector

## Usage

``` r
rxSerialize(x, type = c("xz", "bzip2", "qs2", "qdata", "base"))
```

## Arguments

- x:

  object to serialize

- type:

  serialization type; one of "qs2", "qdata", "base", "xz" or "bzip2".

## Value

raw vector

Matthew L. Fidler
