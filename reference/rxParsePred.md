# Prepare Pred function for inclusion in rxode2

Prepare Pred function for inclusion in rxode2

## Usage

``` r
rxParsePred(x, init = NULL, err = NULL, addProp = c("combined2", "combined1"))
```

## Arguments

- x:

  pred function

- init:

  Initialization vector

## Value

rxode2 transformed text.
